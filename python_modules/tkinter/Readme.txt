tkinter: in python it is standard python GUI library. It gives use object-oriented interface to the GUI toolkit

steps:
import the Tkinter module -> Create the GUI application main window -> Add widgets -> Enter the main event loop

widgets: is an element of a GUI that displayes information or provides a specific way for a user to interact with the operating system or an application
 -> All tkinter widgets will have geometric measurements
 -> Geometric manager classes have 3 methods 
   -> pack() : It organizes the widgets in the block which mean it occupies the entire width
   -> grid() : it organizes the widgets in table-like structure
   -> place() : it is used to place widgets at a specific position you want
